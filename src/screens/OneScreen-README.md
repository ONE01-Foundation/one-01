# OneScreen – מה יש עכשיו vs מה רוצים (לפי התמונות)

## מה יש לנו עכשיו (עדכון)
- **רקע:** צבע תמה + שכבת Background-Ovarly עדינה.
- **טופ/בוטום בר:** גרדיאנט; בדסקטופ הזחה מהקצה. **אזור הכדורים** עם padding כך שהכדורים הקטנים נראים בין הברים.
- **תיבת "עכשיו?":** **קבועה מעל הכדור**, לא בתוך הכדור – לא זזה, הכדור בא אליה. מעל כל השכבות (zIndex 10).
- **כדורים:** **בלי מילוי ובלי מסגרת** (transparent). בכדור קטן: רק **אימוג'י**. בכדור גדול: אימוג'י + כותרת + תת־כותרת + כפתור ONE.
- **מעבר בין כדורים:** **מסך אחד** – רק שלושה מק slots (מעל, מרכז, מתחת). החלפה עם **מחווה swipe** (למעלה/למטה), בלי ScrollView של כמה עמודים – כך שתמיד **כדור קטן אחד למעלה וכדור קטן אחד למטה**, בלי כפילות.
- **אימוג'י לכל תהליך:** למשל 👤 חשבון אנונימי, 🚗 רישיון נהיגה, 💪 עלות 5 קילו מסה.

## מה רוצים (לפי התמונות)
1. **כדור מרכזי = כרטיס לבן מלא** (לא רק מסגרת):
   - **בפנים:** שעה (13:21), תאריך (שלישי, 17.02), כותרת (למשל "רישיון נהיגה"), תת־כותרת (תהליך ל…), שורת "עכשיו?" (מיקרופון + טקסט + פלוס), כפתור ONE.
2. **כדורים קטנים = תצוגת הבא/הקודם** – נראים כ**מציצים** מלמעלה ומלמטה (חצי גלוי, שקופים/מעומעמים).
3. **כל כדור = תהליך/נושא אחר** (רישיון נהיגה, עלות 5 קילו מסה, שינוי טקסט וכו') – גלילה מעבירה את הכדור הבא למרכז והוא גדל.
4. **רקע:** לבן נקי ב-light; Overlay עדין.
