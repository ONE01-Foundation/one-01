# ONE — דף נחיתה

דף נחיתה בסגנון [Phantom](https://phantom.com/): Hero, תכונות, איך זה עובד, אבטחה, CTA ו-Footer.

## איך להציג

- **בדפדפן:** פתח את `index.html` ישירות (לחיצה כפולה או `open landing/index.html`).
- **שרת מקומי:** מתוך תיקיית הפרויקט:
  ```bash
  npx serve landing
  ```
  ואז גלוש ל־`http://localhost:3000`.

## תוכן

- **ניווט:** לוגו ONE, קישורים (תכונות, איך זה עובד, בטיחות, תמיכה), כפתור הורדה.
- **Hero:** כותרת ראשית + תיאור + CTA.
- **תכונות:** Origin, עולמות, תהליכים חיים, סוכן ONE.
- **מקום אחד:** הסבר + רשימת יתרונות.
- **בטיחות:** כרטיסים (נתונים אצלך, תמיכה, מודולרי).
- **הורדה:** לינקים ל־App Store / Google Play (ניתן לעדכן).
- **Footer:** מוצר, משאבים, חברה, רשתות, תנאים ופרטיות.

הדף ב־RTL ועברית, עם פונט Heebo.
